# Motion


## 基础介绍


JavaScript动画库
基于 Web Animations API（WAAPI） 的现代动画库，适用于原生 JavaScript、React、Vue 等框架

## 核心内容
```yaml
motion:
    react:
        motioin:
            button:
            div:
    animate(): # 手动执行动画，支持css选择器
        opt1:
            scale:
        opt2:
            delay:
            duration:
    scroll(): # 滚动监听
    stagger(): # 帧间隔
```
