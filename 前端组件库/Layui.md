# Layui



## 基础介绍

开源 Web UI 组件库
Layui采用模块化加载，需要layui.use()进行初始化





## 核心内容
```yaml
layui js:
    cache: 
    carousel:
        render():
    colorpicker:
        render():
            done():
    element: # 选项卡、进度条
    flow: # 无线加载
    form: # 表单
        on():
    laydate: # 日期选择器
        render():
            elem:
    layer: # 弹窗
        alert():
        close():
        confirm():
        msg():
    table: # 表格
        render():
            cols:
            data:
            elem:
    tree:
        render():
    upload: # 文件上传
        render():
            elem:
            url:
            done():
    config():
    component(): # 定义组件
    data():
    define(): # 定义模块
    device():
    extend():
    link():
    sessionData():
    url():
    use(): # 使用模块

layui css:
    layui-carousel:
    layui-btn: # 按钮
        layui-btn-danger:
        layui-btn-normal:
    layui-form: # 表单
        layui-form-item:
        layui-form-label:
    layui-input: # 输入框
        layui-input-block:
    layui-show:
    layui-tab: # 选项卡
        layui-tab-content:
        layui-tab-item:
        layui-tab-title:
    layui-table: # 表格
    layui-this:
```
