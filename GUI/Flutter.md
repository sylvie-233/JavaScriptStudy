# Flutter

`Flutter官方文档：https://docs.flutter.dev/`


## 基础介绍

基于Dart的跨平台GUI框架

依赖JDK、Android SDK(配置ANDROID_HOME)、Flutter SDK、Android模拟器


Flutter SDK中包含了Dart SDK


资源镜像：
- PUB_HOSTED_URL:
- FLUTTER_STORAGE_BASE_URL:




### 安装目录
```yaml
flutter:
    /.pub-cache: # 下载的公共模块
        /bin:
    /bin:
        /cache:
            /dart-sdk:
                /bin:
                    /resources:
                    /snapshots:
                    /utils:
                    dart:
                /include:
                /lib:
                /pkg:
        /internal:
        /mingit:
        dart:
        flutter:
    /dev:
    /docs:
    /engine:
    /examples:
    /packages:
```
