# git

## 基础介绍

### git
```yaml
git:
    add: # 缓存stash
    branch: # 分支管理
        -d: # 删除分支
    check: # 切换分支
        -b: # 指定分支
    commit: # 提交
        -a: # 所有文件
        -m: # 备注信息
    config:
        --global: 
    merge: # 分支合并
    push: # 推送
        -u: # 远程推送
    remote: # 远程仓库管理
        add: # 添加远程仓库
    stash:  
    status: # 仓库状态
```
