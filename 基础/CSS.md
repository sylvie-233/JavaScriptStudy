### SVG

```yaml
<svg>:
    <g>: 分组
    <defs>: 属性定义
        <filter> 过滤器
            id:
        <lineGradient> 线性渐变
            <stop>
    <rect> 矩形
        x:
        y:
        rx:
        ry:
        width:
        height:
        fill:
        stroke:
        stroke-width:
    <circle> 圆形
        cx:
        cy:
        r:
    <ellipse> 椭圆
    <line> 线
        x1:
        y1:
        x2:
        y2:
    <polygon> 多边形
        points:
    <polyline> 多线条
        points:
    <text> 文本
        <tspan> 行内文本
        transform:
    <path> 路径
        d: 命令
            M: 移动
            L: 绘制直线
            q: 贝塞尔曲线
```
